# github-registrator
1ページ内でリポジトリ[アプリの保管庫]であるGitHubへ一発でアプリを登録できます！
## 🌐 公開ページ
https://github-registrator.vercel.app/
## ✨ 機能
-面倒なCLI操作なく1ページ内でGitHubへの登録が可能となります
-Webアプリならばアプリの公開ページにもそのままデプロイ(利用可能な状態に)します
##⚠️注意
-GitHub及びVercelへのアカウントは事前にGoogleアカウントなどを使用して登録しておく必要があります。
-アクセストークンも先に両サイト分準備しておく必要があります。これは必須です。
-ベーシックインフォのリポジトリネームは、どんなアプリ名にするのか英小文字で決めておきます。ディスクリプションには、ひと言でどんなアプリなのかを説明します。ヴィジビリティはパブリックに公開するのか、もしくはプライベートで個人用にするのかの設定です。ライセンスは、通常ならばMITで良いと思います。知っている方は別なものを選べます。Topcsには、アクセスする方が見つけやすいTagを入れられます。ReadME.MDを自動生成して追加するにチェックを入れられます。
-README CONTENTには、入れておきたい内容を書くと、それを自動生成することができます
-デプロイ・ターゲットはGitHub及びVERCELの両サイトに対して一度に行うことができます。
## 🎵 対応デバイス
PC・スマートフォン（ブラウザのみ、インストール不要）
